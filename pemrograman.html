Materi Pemrograman Web

Berisi contoh program HTML, CSS, JS, PHP.

  • HTML

    • HTML adalah singkatan dari Hyper Tect Markup Language. HTML adalah bahasa markup standar untuk membuat halaman Web.

  • CSS

    • CSS merupakan singkatan dari Cascading Style Sheet. CSS biasanya digunakan untuk mengubah atau mengatur tampilan halaman web.

  • JS

    • JS merupakan singkatan dari Javascript. JS digunakan untuk membuat halaman web interaktif.

  • PHP

    • PHP adalah salah satu bahasa pemrograman skrip yang dirancang untuk membangun aplikasi web.Pemrosesan program PHP dilakukan di lingkungan web server.
